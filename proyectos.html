Proyecto Crud 4

CRUD es un acrónimo que representa las cuatro operaciones básicas que se pueden realizar en una base de datos o sistema de almacenamiento de datos: Create (Crear): Añadir nuevos registros a la base de datos. Read (Leer): Recuperar datos existentes de la base de datos. Update (Actualizar): Modificar datos existentes en la base de datos. Delete (Eliminar): Borrar registros de la base de datos.
